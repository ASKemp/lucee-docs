id: function-dayofweek
related:

---

Returns the integer index of the day of the week for the given date. The index starts at 1 for _sunday_ and continues up to 7 for _saturday_. i.e.

* 1 = Sunday
* 2 = Monday
* 3 = Tuesday
* 4 = Wednesday
* 5 = Thursday
* 6 = Friday
* 7 = Saturday
